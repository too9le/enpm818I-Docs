The \ThisSys \ThisSegment \DTV capability is divided into the following specification groups:
\begin{description}
	%\item[Output processing] provides the \DTV broadcast output waveform processing capability, \S~\ref{loc:Waveform_DTV_Output}.
	%\item[Input processing]  provides the \DTV broadcast input waveform processing  capability, \S~\ref{loc:Waveform_DTV_Input}.
	\item[Transmission processing] provides the \DTV broadcast transmission waveform processing capability.%, \S~\ref{loc:Waveform_DTV_Transmission}.
	\item[VSWR processing] provides the \DTV broadcast VSWR processing capability.%, \S~\ref{loc:Waveform_DTV_VSWR}.
\end{description}
